# Tello Drone Web Controller

WiFiバインド対応・LineTrace搭載のTelloドローン制御Webアプリケーション

A web-based DJI Tello drone controller with WiFi interface binding, LineTrace, and a responsive dark-mode UI.

## Quick Start

### Windows

```
start.bat
```

### macOS / Linux

```bash
chmod +x start.sh
./start.sh
```
